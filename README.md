## Развертывание
#### Установка библиотек
```
npm install
```
#### Команды для компиляции
_Запустить webpack-dev-server:_  
```
npm run start
```  
_Загрузить на gh-pages:_  
```
npm run deploy
```  
_Скомпилировать код в режиме development:_  
```
npm run dev
```  
_Скомпилировать код в режиме production:_  
```
npm run build
```
